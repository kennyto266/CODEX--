#### Docker多阶段构建
```dockerfile
# 构建阶段
FROM node:18-alpine AS builder
# 构建前端...

# 生产阶段
FROM python:3.10-slim
# 优化生产环境...
```

```yaml
jobs:
  code-quality:     # 代码质量检查

  build-and-push:   # 构建镜像
```
