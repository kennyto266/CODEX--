# 啟動 Bot 指南

## 快速啟動

### 1. 設置環境變量
```bash
python set_env.py
```

### 2. 啟動 Bot
```bash
cd src/telegram_bot
python telegram_quant_bot.py
```

### 3. 在 Telegram 中測試

發送以下命令到 @penguinai_bot：

```
/score
/score nba
/schedule
/favorite Lakers
```
