## 📋 **5分钟快速测试**

### **步骤1: 打开网页 (10秒)**
```
🌐 打开浏览器访问:
http://localhost:8001/task-board-execution.html

✅ 应该看到: 任务看板界面
```

### **步骤2: 添加任务 (30秒)**

点击页面顶部的 **"➕ 添加任务"** 按钮

填写表单:
```
任务ID: TASK-QUICK-001
任务标题: Quick Test Task
描述: Testing web add task feature
优先级: P1
预计工时: 8
```

点击 **"创建任务"** 按钮

### **步骤3: 验证添加 (20秒)**

✅ 模态框自动关闭
✅ 显示成功提示: "任务 TASK-QUICK-001 创建成功！"
✅ 任务列表自动刷新
✅ 新任务出现在 "⏸️ 待开始" 列

### **步骤4: 执行任务 (30秒)**

找到刚创建的 `TASK-QUICK-001`
点击任务旁的 **🚀** 按钮

观察:
- 任务移至 "⚡ 执行中"
- 显示执行输出
- 任务标记为 "✅ 已完成"

### **步骤5: 查看结果 (10秒)**

点击 **"🔄 刷新"** 按钮
确认任务状态已更新

**总计用时: ~2分钟** ⏱️

## 🔍 **故障排除**

### 问题1: 按钮不显示
**原因**: 前端未更新
**解决**: 刷新页面 (Ctrl+F5)

### 问题2: 创建失败
**原因**: API未启动或端口错误
**解决**:
```bash
# 检查API状态
curl http://localhost:8000/tasks

# 重新启动API
python simple_task_api.py &
```

### 问题3: 任务不显示
**原因**: 浏览器缓存
**解决**:
1. 硬刷新页面 (Ctrl+Shift+R)
2. 检查开发者工具 (F12) 控制台
