## 💡 **策略原理与机制**

### Z-Score均值回归策略
**核心思想**: HIBOR利率围绕长期均值波动，偏离均值后有回归动力

**信号生成**:
```python
# 计算Z-Score
zscore = (hibor - sma15) / std15

# 交易信号
if zscore < -0.5:
    action = "BUY"  # HIBOR严重低估
elif zscore > 0.5:
    action = "SELL" # HIBOR严重高估
else:
    action = "HOLD" # 中性区间
```

**为什么高频有效**:
1. **短期波动**: HIBOR每日都有微小波动，0.5阈值捕捉更多机会
2. **快速回归**: 利率偏离后通常1-3天内回归
3. **复利效应**: 更多交易 = 更多复利增长
4. **风险分散**: 94次交易分散风险，单次损失影响小
