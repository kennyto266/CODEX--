## 📊 系統架構

```
LIHKG 爬蟲系統
├── 核心模組
│   ├── scraper.py         (爬蟲引擎)
│   ├── sentiment.py       (情緒分析)
│   ├── stock_identifier.py (股票識別)
│   └── storage.py         (數據存儲)
├── API 服務
│   └── routes.py          (RESTful API)
├── 數據存儲
│   └── demo_run.db        (SQLite 資料庫)
└── 測試套件
    ├── demo_run.py        (演示程序)
    ├── quick_test.py      (快速測試)
    └── tests/             (測試文件)
```

## 🚀 使用方法

### 運行爬蟲
```bash

python main.py
```

