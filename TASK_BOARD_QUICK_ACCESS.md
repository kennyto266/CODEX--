## 🚀 快速操作

### 查看任務統計
訪問任一頁面即可看到：
- 總任務數
- 已完成數量
- 進行中數量
- 已阻塞數量
- 完成率百分比

### 任務看板功能
- **拖拽任務**: 雙擊任務或點擊箭頭按鈕
- **狀態流轉**: 待開始 → 進行中 → 待驗收 → 已完成
- **優先級**: P0(紅色) / P1(黃色) / P2(藍色)
- **統計卡片**: 實時顯示進度

## 📝 任務數據概況

### 按狀態分組
- **待開始**: 大量任務
- **進行中**: 多個核心任務
- **待驗收**: 部分已完成任務
- **已完成**: 少量任務 (1個)
- **已阻塞**: 少數任務

### 按優先級分組
- **P0 (關鍵)**: 大量任務
- **P1 (重要)**: 中等數量
- **P2 (一般)**: 少數任務

## ⚡ 使用提示

1. **任務流轉**: 雙擊任務卡片或使用箭頭按鈕

## 🆘 故障排除

### 頁面無法訪問
```bash
# 檢查服務狀態
ps aux | grep python | grep http.server
# 如果沒有運行，啟動服務：
cd src/dashboard/static
python -m http.server 8001
```

### API連接失敗
```bash
# 檢查API服務
curl http://localhost:8000/tasks
# 如果沒有運行，啟動：
python simple_task_api.py
```

### JavaScript錯誤
- 清除瀏覽器緩存 (Ctrl+F5)
- 確保使用正確的URL (http://localhost:8001)
- 檢查瀏覽器控制台錯誤
