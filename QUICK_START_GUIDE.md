# 香港市場量化分析 - 快速開始指南

## 一分鐘摘要

我們對香港政府替代數據和HKEX市場數據進行了深入的量化分析，主要發現：

1. **6個月HIBOR對市場具有統計顯著的預測能力**（Granger因果性p<0.05）
2. **市場顯示正Alpha（12.81%年化）**
3. **開發了5個可交易的策略框架**

## 最重要的發現

### 6個月HIBOR的預測能力

**Granger因果性檢驗結果**:
- 1天滯後: p=0.017 ✓
- 2天滯後: p=0.015 ✓
- 3天滯後: p=0.018 ✓
- 4天滯後: p=0.018 ✓
- 5天滯後: p=0.049 ✓

**結論**: 6個月HIBOR對未來1-5天市場收益具有統計顯著的預測能力

### 其他關鍵相關性

- **12個月HIBOR vs 市場**: +0.257（正相關）
- **交易筆數 vs 市場**: -0.397（恐慌指標）
- **12M HIBOR滯後2天**: -0.374（p=0.038，顯著）
